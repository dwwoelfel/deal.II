Downloading deal.II

deal.II is Open Source and therefore freely available. You can download the following files (save the contents of the links into a separate file):

  • deal.II-3.1.1.tar.gz: deal.II Version 3.1.1

  • deal.II-3.1.0.tar.gz: deal.II Version 3.1.0

  • deal.II-3.0.1.tar.gz: deal.II Version 3.0.1

  • deal.II-3.0.0.tar.gz: deal.II Version 3.0.0

For instructions on unpacking and installation, see the README file. Announcements and news for the various releases can be found on the news page.
