Publications on and with deal.II

On this page, we would like to collect a list of as many publications written on or with the help of deal.II as possible. If possible, we will cross-reference them to the author for questions and possible downloads.

Given the young age of deal.II, the present list of publications is still relatively small. However, several more publications are presently in progress. If you have written something, we would be pleased if you could drop us a brief note (if possible including abstract and/or BibTeX entry), so that we can add your entry to the list. We are interested in listing applications in as many fields as possible.
