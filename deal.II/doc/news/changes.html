Changes after Version 5.2

This is the list of changes made after the release of deal.II version 5.2. It is subdivided into changes made to the three sub-libraries base, lac, and deal.II, as well as changes to the general infrastructure, documentation, etc.

All entries are signed with the names of the author. Regular contributor's names are abbreviated by WB (Wolfgang Bangerth), GK (Guido Kanschat), RH (Ralf Hartmann).

General

  1. Fixed: The PETSc libraries have been relinked each time make was called. Now, the PETSc libraries will only be relinked if necessary.
    (RH 2005/09/15)

base

  1. New: There are now new get_n_mpi_processes and get_this_mpi_process functions in the Utilities::System namespace. In case of a codes not running in parallel, they simply return 1 and 0, respectively.
    (RH 2005/09/20)

lac

  1. New: Now the class SparseILU supports also the Tvmult method. This feature allows the straightforward use of SparseILU as a multigrid smoother within the framework.
    (Oliver Kayser-Herold 2005/09/15)

deal.II

  1. Fixed: The GridIn<dim>::read(filename, format) function ran into an exception when called with Default format and a filename which does not include /. This is now fixed.
    (RH 2005/09/21)
